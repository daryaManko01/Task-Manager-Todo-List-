# ToDo List

A simple task management application built with Vanilla JavaScript.

## Features

* Add new tasks
* Mark tasks as completed
* Add tasks to favorites
* Move tasks between active, completed, and favorite lists
* Automatic counters for completed and favorite tasks

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```text
├── ToDo List.html
├── ToDo.js
└── src/
```

## Getting Started

Clone the repository and open `ToDo List.html` in your browser.

```bash
git clone https://github.com/your-username/todo-list.git
```
